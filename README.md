# Update Locker

Block updates (and auto-updates) selected apps via popular markets
Block apps permissions revoke (system-wide)

## List of supported markets:
- Google Play Market
- Aurora Store
- Huawei AppGallery
- Samsung Galaxy Store
- Steam
- F-Droid
- App Lounge
- Aptoide
- RuMarket
- RuStore
- Mi GetApps
- Xiaomi Market
- Palm Store
- M Store (ex. Meizu App Store Global)
